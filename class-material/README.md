# Class Materials

Drop your course materials here. The research agents search this directory for quotes and evidence.

## Supported Formats

- **Markdown (.md)** — Best for typed lecture notes. Directly searchable.
- **Text (.txt)** — Directly searchable.
- **PDF (.pdf)** — Run `bash scripts/convert-pdfs.sh` to convert to searchable text in `converted/`.

## Naming Conventions

| Type | Format | Example |
|---|---|---|
| Lecture notes | `lecture-XX-topic.md` | `lecture-03-realism.md` |
| Readings | `author-year-short-title.md` | `waltz-1979-theory-intl-politics.md` |
| Textbook chapters | `textbook-chXX-topic.md` | `textbook-ch05-liberalism.md` |
| PDFs | Any descriptive name | `mearsheimer-2001-tragedy.pdf` |

## Tips

- **Include page numbers** in typed notes when possible (e.g., `(p. 42)`) — line numbers in converted text won't match PDF pages
- **One reading per file** makes citations cleaner
- **Keep originals** — the `converted/` folder is auto-generated and can be re-created anytime
